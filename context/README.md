# /context — materiales fuente del estudio

Carpeta de trabajo para reunir y ordenar documentación dispersa de Studio Nomade
(planillas Excel, PDF de presupuestos, SLA, plantillas, procesos) **antes** de
estructurarla dentro de Noma (módulo Contexto/Documentación).

- El contenido de esta carpeta **no se versiona** (ver `.gitignore`): puede contener
  información sensible o privada del estudio.
- Úsala como bandeja de entrada: sube aquí los documentos exportados desde Google Drive,
  revísalos y luego cárgalos/estructúralos en la plataforma.
- Sugerencia de organización: subcarpetas por categoría (`presupuestos/`, `sla/`,
  `procesos/`, `plantillas/`, `referencia/`).
